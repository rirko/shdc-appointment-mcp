When adding cases, append to this file using the same Markdown structure. Prefer concrete user language and department keywords.

## Case: 膝关节疼痛-运动后加重

- 用户原话：膝盖疼，走路久了更明显。
- 初始模糊点：部位不清、持续时间不清、是否外伤不清。
- 推荐追问：
  1. 疼痛具体在膝盖前方、内侧、外侧还是后方？
  2. 持续多久？是否有扭伤或撞击？
  3. 是否红肿、发热、不能负重？
- 关键线索：
  - 运动后加重、无急性外伤：考虑骨科/运动医学科。
  - 不能负重、明显肿胀或畸形：急诊/骨科急诊。
- 建议科室：骨科、运动医学科。
- 推荐挂号类型：专病；如果医院没有运动医学/关节专病号，则查专家。
- MCP 查询关键词：骨科、运动医学、关节、膝。
- 相似案例判断标准：下肢关节痛，活动相关，无明确全身急症表现。
- 不能做的结论：不能直接诊断半月板损伤、韧带撕裂或关节炎。

## Case: 胸口压榨痛-伴出汗

- 用户原话：胸口像被压着，还出汗，有点喘。
- 初始模糊点：持续时间、年龄和既往病史不清，但危险信号已足够。
- 推荐追问：
  1. 不继续普通追问；直接建议急诊/120。
- 关键线索：
  - 胸痛伴出汗、气短：心血管急症风险。
- 建议科室：急诊。
- 推荐挂号类型：不走普通预约；建议急诊。
- MCP 查询关键词：不做普通门诊查询。
- 相似案例判断标准：胸痛伴出汗、气短、晕厥、放射痛任一组合。
- 不能做的结论：不能在聊天中排除或确诊心肌梗死。

## Case: 咳嗽发热-三天

- 用户原话：咳嗽三天，有点发烧。
- 初始模糊点：体温、呼吸困难、基础病、核酸/流感检测不清。
- 推荐追问：
  1. 最高体温多少？有没有明显喘不上气或胸痛？
  2. 咳痰吗？痰是什么颜色？有没有咯血？
  3. 是否有老人、孕妇、慢性肺病或免疫低下情况？
- 关键线索：
  - 普通咳嗽发热：呼吸内科/发热门诊。
  - 严重呼吸困难、胸痛、意识异常：急诊。
- 建议科室：呼吸内科、发热门诊。
- 推荐挂号类型：专家；若已有明确哮喘、慢阻肺、肺结节等诊断，可查专病。
- MCP 查询关键词：呼吸内科, 发热门诊, 咳嗽。
- 相似案例判断标准：急性呼吸道症状，以咳嗽/发热为主。
- 不能做的结论：不能仅凭描述区分普通感冒、流感、新冠或肺炎。

## Case: 腰痛-伴腿麻

- 用户原话：腰疼，还麻到腿。
- 初始模糊点：是否外伤、麻木范围、肌力、大小便功能不清。
- 推荐追问：
  1. 有没有一侧腿无力、走路拖脚或大小便控制异常？
  2. 腰痛多久了？是否突然发生或有外伤？
  3. 麻木是从腰臀部放射到腿，还是局部皮肤麻？
- 关键线索：
  - 腰痛伴放射腿麻：骨科/脊柱外科，也可神经内科。
  - 大小便障碍、进行性无力：急诊。
- 建议科室：骨科、脊柱外科、神经内科。
- 推荐挂号类型：专家；若已有明确腰椎间盘突出/脊柱疾病随访，可查专病。
- MCP 查询关键词：骨科, 脊柱, 神经内科, 腰椎。
- 相似案例判断标准：腰背痛伴下肢放射痛或麻木。
- 不能做的结论：不能直接诊断腰椎间盘突出或椎管狭窄。
