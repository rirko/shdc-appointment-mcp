# Hospital WeChat and Self-registration Links

Use this after hospital/slot lookup and before formal booking. If the user has not completed offline first visit/hospital registration, or wants to register by themselves, attach the matching hospital link.

Source: local document `上海市主要医院微信服务信息.docx` in the project workspace.

## Use Rules

- Match by hospital short name, full name, or common alias.
- Do not invent links. If a hospital is not listed here, say no matching link is currently recorded.
- If the selected hospital has multiple branches in MCP output, provide the link only when the hospital identity is clear; otherwise ask the user to choose the exact hospital/branch first.
- These links are user-facing convenience links. They do not replace emergency advice and do not bypass the MCP safety rules.
